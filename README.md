# ANA 웹사이트 2026

2026년에 개발한 ANA 동아리 웹사이트 입니다. 

## 소개

충남대학교 컴퓨터인공지능학부 소속 알고리즘 문제해결 동아리, ANA의 웹사이트 입니다.\
기존에 있던 페이지와는 다르게 게시판, 갤러리, 오늘의 문제 등 회원들과 의사소통을 할 수 있는 다양한 기능을 넣었습니다. (많이 사용할지는 모르겠네요...ㅎㅎ)\
자세한건 아래에서 확인해주세요!

### About
기존에는 동아리 소개를 member, about, activity 등 다양한 페이지로 나누었는데, 이번엔 About 한 페이지로 합쳤습니다! 이로 인해 페이지가 너무 길어진다는 단점이 있는데요, 이는 사이드 네비게이션으로 어느정도 해결했습니다. 게시판이나 입부신청 페이지 같이 페이지가 너무 많아져서 복잡해질바엔 하나로 합치는게 나을 것 같았습니다. 이외에도 좀 더 ana스럽게 하기 위해 '물마루'라는 8bit 폰트를 사용했으며, 사진을 좀 더 잘 보여주기 위해 'swiper'라는 패키지를 사용했습니다.

### 오늘의 문제
홈 화면 오른쪽 구석에 있는 오늘의 문제, 어찌보면 알고리즘 문제해결 동아리에서 가장 중요하다고 생각해서 의외로 엄청난 노력을 들여서 만들었습니다. 처음엔 백준 크롤링을 하려고 했는데, 백준은 크롤링을 막기도 하고 랜덤 문제의 조건을 설정하지 못하기 때문에 솔브닥으로 옮겼습니다. 하지만 솔브닥은 봇이라고 판단하면 가차없이 막아버리기 때문에, 결국 제 컴퓨터에서 문제를 검색하여 파일로 저장한 다음, 서버에서 랜덤으로 하루에 세 개씩 뽑아서 올리는 방법으로 변경하였습니다. 문제 범위는 '티어: 브론즈5-골드1', 푼 사람 수: 5000-∞' 으로 설정하여 많은 사람들이 쉽고 재밌게 도전해볼 수 있도록 하였습니다.

### 게시판
자유게시판, 공지사항, 알고리즘 꿀팁 게시판이 있습니다. 에디터는 'ckeditor' 패키지를 사용하여 좀 더 쉽고 원활하게, 전문적?으로 작성할 수 있도록 하였습니다. 사진 첨부는 'multer' 패키지를 사용하여 가능하게 했습니다. 물론 댓글 기능도 있습니다. 자유롭게 소통해보세요!

### 갤러리
게시판이랑 비슷한데 사진을 좀 더 집중적으로 올리는 곳으로, 그냥 인스타그램 피드 정도라고 생각하시면 됩니다. 인스타처럼 만들고 싶었는데, 자유게시판이랑 기능이 겹쳐서 없앨까 고민중입니다. 근데 생각보다 잘 만들어버려서 아직은 보류하겠습니다.

### 학과공지
컴퓨터융합학부 사이트에 올라오는 공지 가져와서 보여주는 페이지 입니다. [우리 황현석 회장님의 cnu-noticer](https://github.com/Hy3ons/cnu-noticer)를 적극 사용하였습니다. 회장님의 디비에서 데이터를 가져와서 'react-markdown'으로 보여줍니다. 사진이랑 첨부파일도 보여줘야 하는데, 귀찮아서 이 기능은 보류해놨습니다. 보고싶으신 분들은 [https://cnu-noticer.vercel.app](https://cnu-noticer.vercel.app) 여기서 확인 부탁드립니다.

### 그 외 자랑할 것들
- 모바일도 잘 볼 수 있도록 엄청나게 노력했습니다. 반응형을 써본 적이 처음이라 고생 많이 했습니다. (운동 많이 된다. 이런 스트레스도 필요해)
- 구글, 깃허브 로그인이 가능합니다. 멋지죠?
- 도커 파일들을 야무지게 작성해서 어느 서버에도 올릴 수 있습니다.
- ci cd 구축해서 깃헙에 push만 해도 자동으로 배포됩니다.

## 기술 스택

### 언어
* `typeScript`

### 프론트엔드
* `react` 컴포넌트 기반 라이브러리
* `vite` 빌드 툴
* `zustand` 전역 상태 관리
* `react router` 라우팅
* `tailwind css` css 유틸리티
* `daisy ui` 예쁜 ui
* `ckeditor` 에디터
* `motion` 애니메이션
* `swiper` 사진 캐러셀
* `axios` 서버랑 통신
* `dompurify` xss 방지

### 백엔드
* `node js` 런타이 
* `express` 웹 프레임워크
* `passport js` 구글, 깃헙 로그인 하게 해준거
* `jsonwebtoken` 인증 및 인가 
* `mongo db` 데이터베이스
* `mongoose` js에서 데베 쓸 수 있게 해주는거
* `express mongo sanitize` nosql injection 막아주 
* `multer` 사진 업로드
* `bcrypt` 비밀번호 해시화 및 솔팅, 키 스트레칭
* `cors` 다른 도메인 막아줌
* `express validator` 유효성 검사
* `helmet` 헤더 보안
* `node cron` 스케쥴러
* `cookie parser` 쿠키 읽을 수 있음
* `nanoid` id 생성기

### 인프라
* `docker` 컨테이너
* `github actions` ci cd
